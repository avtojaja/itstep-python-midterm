# თარჯიმანის აპლიკაცია

თარჯიმანის აპლიკაცია მომხმარებელს საშუალებას აძლევს აირჩიოს ენების წყვილი - რომელი ენიდან რომელ ენაზე უნდა ითარგმნოს სიტყვები (მაგ: `ქართული-ინგლისური`, `ქართული-ესპანური`, `ინგლისური-ქართული`, `ესპანური-ქართული`). ამის შემდეგ მომხმარებელს შეჰყავს **სიტყვა** ან **მოკლე ფრაზა**. პროგრამა კითხულობს თარგმანებს `dictionary.json` ტექსტური ფაილიდან (ლექსიკონიდან) და აჩვენებს შესაბამის ნათარგმნ სიტყვას ან ფრაზას. იმ შემთხვევაში, თუ სათარგმნი სიტყვა არ აღმოჩნდა ლექსიკონში, პროგრამა მომხმარებელს სთავაზობს დაამატოს ამ ახალი სიტყვის ან ფრაზის თარგმანი ლექსიკონს (ტექსტურ ფაილს). შეცვალეთ `dictionary.json` ტექსტური ფაილი, რომ დაამატოთ ან შეცვალოთ თარგმანები საჭიროებისამებრ.

### პროექტი აკმაყოფილებს შემდეგ კრიტერიუმებს:
1. პროექტში გამოყენებული უნდა იყოს სხვადასხვა ტიპის ცვლადები.
2. პროექტში გამოყენებული უნდა იყოს პირობითი და ციკლის ოპერატორები.
3. პროექტში გამოყენებული უნდა იყოს მასივები.
4. პროექტში სტუდენტს შექმნილი უნდა ჰქონდეს სამომხმარებლო ფუნქციები.
5. სტუდენტმა უნდა გამოამჟღავნოს ფაილების შექმნის/რედაქტირების/წაშლის უნარები.
6. პროექტში სტუდენტმა უნდა გამოამჟღავნოს OOP-ის პრინციპების ცოდნა (მემკვიდეობა, ინკაფსულაცია, პოლიმორფიზმი).
7. კოდის ყველა მოდული დაკომენტარებული უნდა იყოს.
8. აპლიკაციის გამოყენებისას უზრუნველყოფილი უნდა იყოს შეყვანის სათანადო ვალიდაცია და შეცდომების დამუშავება.
